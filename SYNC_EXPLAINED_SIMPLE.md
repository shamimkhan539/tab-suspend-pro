## 📍 What Does "Where You're Signed In" Mean?

### Simple Version

**Any computer where you log in to Chrome with your Google account**

```
Example: Your Google Account is: shamim@gmail.com

Device 1: Sign in to Chrome with shamim@gmail.com
  └─ Extension syncs here ✓

Device 2: Sign in to Chrome with shamim@gmail.com
  └─ Extension syncs here ✓

Device 3: Sign in to Chrome with shamim@gmail.com
  └─ Extension syncs here ✓

Device 4: Sign in to Chrome with different@gmail.com
  └─ ✗ Won't sync (different account)
```

## 🔄 How Sync Works (Simple Flow)

```
Device A (Home)
├─ Sign in: shamim@gmail.com
├─ Toggle: ON ✓
├─ Settings saved
└─ chrome.storage.sync → Google Cloud → Encrypted
       ↓
       (Chrome handles this automatically)
       ↓
Device B (Work)
├─ Sign in: shamim@gmail.com
├─ Chrome: "Found your extension settings!"
├─ Download from cloud → Chrome decrypts
└─ Toggle automatically: ON ✓
```

## 🎯 Step by Step: Enable Sync

### Step 1: Check You're Signed Into Chrome

```
Chrome Menu (⋮) → Sign in to Chrome
│
├─ NOT signed in? → Click "Sign in" → Use your Google account
│
└─ Already signed in? → Skip to Step 2
```

### Step 2: Open Extension Options

```
Extension Icon → Manage extensions
→ Click extension name
→ Options / Settings
```

### Step 3: Toggle Sync

```
Look for: "✨ Sync Across Devices (Recommended)"

Click checkbox: ☐ → ☑

Result: See message "Chrome sync enabled!"
```

### Step 4: Verify on Another Device

```
Device 2:
├─ Same Google account
├─ Open extension options
├─ You should see: ☑ (already ON)
└─ No setup needed - it's automatic!
```

## ❓ FAQ: Quick Answers

### Q: Will my passwords sync?

**A**: Only the settings this extension uses. Not browser passwords.

### Q: Is it secure?

**A**: Yes! Google encrypts everything. Only you can decrypt it with your account.

### Q: What if I lose internet?

**A**: Settings stay on your device. Syncs when back online.

### Q: Can I turn it off?

**A**: Yes! Toggle it OFF. Settings stay local only. Use Export/Import instead.

### Q: Does it cost anything?

**A**: No! It's part of Chrome Sync, included free with Chrome.

### Q: How fast does it sync?

**A**: Usually within seconds to minutes, depending on internet.

### Q: Works on Chromebook?

**A**: Yes! Any Chrome installation on any device.
